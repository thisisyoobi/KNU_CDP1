SFU Server Support 1: N connection 

- 시연을 위해 LocalHost 환경에서만 작동합니다. 
- 자신의 화면을 공유하여 여러명의 Client에게 스트리밍할 수 있습니다.
- 주의 ! https:// 에서 사용해 주십시요.

## How To Install
```sh
npm install

npm run dev

https://localhost:3000/admin ==> 스크린을 공유하는 역할 
https://localhost:3000 ===> 스크린을 공유받는 역할
```

## Important
- https:// 환경에서 사용해 주십시요.
- 공유할 창을 최소화하지 말아주십시요.
- port 3000번을 비워주십시요.
